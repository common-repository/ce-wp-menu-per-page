Provides a selectbox for menues while editing pages.

== Description ==

Adding a small widget box to the editing pages section to select a custom menu which is displayed on this page.

**Translations**

* English
* German

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'CE WP-Menu per Page'
1. Click 'install'

**Manual Installation**

1. Upload folder `ce-wp-menu-per-page` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin in your widget-settings
